# PerfectPutt
Perfect putt is a autonomous putter that we wil implement in our senior design project at the University of Florida.


Completed Work:

User application is running and connected to android phone
The user application can succesfully connect to and read data from the Portenta
The user application can succesfully display user feedback based on their swing
Portenta can succesfully detect the ball with a neural network
Portenta can succesfully interface with IMU
Neural Network set up on edge impulse
Container is completed and fits on the putter
Auto putter circuit is finished, solenoid is broken.
auto putter has completed print that fits the head of the putter.



Known Bugs

CNN has F1 ratio of 84%
The Portenta will sometimes shut off and needs to be reconfigured, this is a known portenta issue
BLE connections are by their nature volatile
The product will overheat and brick if left in practice mode for 90+ minutes

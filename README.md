# PerfectPutt
Perfect putt is a autonomous putter that we wil implement in our senior design project at the University of Florida.


Completed Work:

User application is running and connected to android phone
Portenta Can succesfully detect golf balls and holes and do rudimentary distance calculations
Portenta can succesfully interface with IMU
Neural Network set up on edge impulse
Container is completed and fits on the putter
Research on auto putter completed (shipping is extremely delayed)

Known Bugs

CNN has F1 ratio of 84%
The Portenta will sometimes shut off and needs to be reconfigured (overheated)
BLE connections are by their nature volatile
This hasn't been tested on the actual test green (shipping)
The power bank is currently stuck in the slot
